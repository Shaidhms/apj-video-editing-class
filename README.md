# APJ Video Editing Class

A hands-on three-day video editing class deck — interactive HTML presentation built for APJ Academy.

**Live deck**: https://shaidhms.github.io/apj-video-editing-class/

## What's covered

- **Day 1** — Foundations & first cut: video roll, audio, voice-over → first 30-sec reel exported
- **Day 2** — Polish & submit: transitions, masking, effects, keyframes, subtitles → final reel submitted
- **Day 3** — Review & AI tools: live feedback session + Google Flow + Meta AI

10 lessons, 36 slides, all running on free CapCut.

## Controls

- `←` `→` arrows — navigate
- `M` — toggle slide menu
- `F` — fullscreen
- `Esc` — close menu / exit fullscreen
